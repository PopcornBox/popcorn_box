# 영화 평점 웹 사이트 Spring 프로젝트 - POPCORN BOX

리드미 파일 추후 수정 예정입니다.

## 1. 프로젝트 목표
영화 정보와 평점을 한곳에서 확인할 수 있는 사이트입니다. 실시간으로 영화 평점을 반영하여, 영화 순위 차트를 갱신합니다. 그 외에도 커뮤니티와 이벤트 공간을 마련합니다.
